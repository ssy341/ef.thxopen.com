### 美音发音技巧 —— 美式双元音aoww

{% include Audio.html src="/assets/media/pronunciation/day6.mp3" type="mp3" %}

双元音aʊ，第一个音是/a/音：/a/；第二个音是"短乌音"：/u/. 然后由第一个音向第二个音滑动，发音时口形由大到小，发音由长到短，
由清晰到含糊：/aʊ/， /aʊ/ 。美式变音æʊ，与/aʊ/区别就在于前元音好像变成了æ一样，开口更大。
